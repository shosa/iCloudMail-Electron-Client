# Resources

Place these files here before building:

- `icon.ico` — Windows app icon (256×256 ICO). Convert `tray-icon.svg` with a tool like
  https://svgtopng.com/ then https://convertio.co/png-ico/
- `tray-icon.png` — 32×32 PNG for the system tray (convert tray-icon.svg)

The SVG source is `tray-icon.svg`.
